# Operational Runbooks

Step-by-step guides for common operational tasks and incident response.

## Template

- **Title**: What this runbook covers
- **Symptoms**: How to identify when this runbook is needed
- **Steps**: Numbered, exact commands to run
- **Verification**: How to confirm the issue is resolved
- **Escalation**: When and who to escalate to

## Common Runbooks

- Deployment rollback procedure
- Database migration recovery
- Service health check and restart
- Secret rotation procedure
