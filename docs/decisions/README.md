# Architecture Decision Records

This directory contains ADRs (Architecture Decision Records) following the format:

## ADR Template

- **Title**: Short noun phrase
- **Status**: Proposed | Accepted | Deprecated | Superseded
- **Context**: What is the issue that we're seeing that is motivating this decision?
- **Decision**: What is the change that we're proposing/making?
- **Consequences**: What becomes easier or harder because of this change?

## Naming Convention

Number sequentially: `001-feature-name.md`, `002-another-decision.md`, etc.
